Claude Code가 Redmine 일감을 직접 다루도록 MCP 서버를 만들었습니다. 클라우드 커넥터 없이 내 PC에서 로컬로 실행되고, API 키와 주소만 있으면 어떤 Redmine에도 연동됩니다. 제작 기간은 약 1주일.

📦 설치 (어느 OS든 두 줄)
pipx install git+https://github.com/junstellar/redmine-mcp-jun.git
redmine-mcp-setup
